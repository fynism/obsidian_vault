# 路线
### 四、 怎么系统学习？（学习路径建议）

你现在处于“跳过基础学框架”的阶段，建议按以下顺序补课，不要硬啃项目：

#### 第一阶段：补全“地基” (不要超过3天)

- **Servlet/JSP：** 看视频理解请求、响应、Session、Cookie 的原理（不用深究怎么写页面）。
    
- **HTTP 协议：** 了解 GET、POST、状态码（200, 404, 500）。
    

#### 第二阶段：深度学习 Spring 全家桶 

- **Spring IOC & AOP：** 这是 Spring 的灵魂，理解“为什么不需要手动 new 对象”。
    
- **Spring MVC：** 重点学习注解 @RequestMapping, @RequestParam, @RequestBody, @PathVariable。这些是 Controller 层的核心。
    
- **MyBatis/MyBatis-Plus：** 学习 Mapper 接口怎么通过 SQL 操作数据库。
    

#### 第三阶段：设计模式与规范

- 学习什么是 **POJO, DTO, VO, BO**（这是企业开发约定俗成的规范）。
    
- 看一眼 **“MVC 设计模式”**。
